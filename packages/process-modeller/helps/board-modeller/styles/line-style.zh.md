# 线型

  - 厚度：设置线的厚度。
  - 颜色：设置线的颜色。
  - 线的样式：设置线的样式。（直线、虚线等）
  - 线端的样式：设置线端的样式。（square：方形线端，round：圆形线端）
  - 连接样式：设置线与线相连接的点的样式。  
    (锐角相交 : ![miter][miter], 弧形 : ![round][round], 斜角相交 : ![bevel][bevel])
  - 开始形状：设置线的开始形状。
  - 开始大小：设置线的开始大小。
  - 结束形状：设置线的结束形状。
  - 结束大小：设置线的结束大小。
